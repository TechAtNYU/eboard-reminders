# Event Reminders

## Installation

### Install Dependencies
```
virtualenv .venv
. .venv/bin/activate
pip install -r requirements.txt
```

### Environment Variables (Virtual Enviroment Recommened)

```
$SLACK_BACK_BOT_TOKEN
$TNYU_API_KEY
```
